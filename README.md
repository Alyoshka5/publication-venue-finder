# Publication Venue Finder

## How To Run

### Cloning the repository
In the terminal, run:
1. `git clone git@github.com:Alyoshka5/publication-venue-finder.git`
2. `cd publication-venue-finder`

### Running the app
1. Create two terminals (at the root of the /publication-venue-finder directory)
2. In the first terminal, run: `cd client` and `npm run dev`
3. In the second terminal, run: `cd server` and `npm run dev`
